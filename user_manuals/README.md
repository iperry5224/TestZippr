# SAELAR & SOPRA User Manuals

User manuals for ISSO (Information System Security Officer) personnel. Written for end users—not architects or developers.

## Files

| File | Description |
|------|-------------|
| **SAELAR_User_Manual.pptx** | SAELAR quick reference: assessment workflow, Chad AI, SSP, BOD 22-01 |
| **SOPRA_User_Manual.pptx** | SOPRA quick reference: CSV workflow, ISSO Toolkit, Vulnerability Management |

## Google Slides

1. Open [Google Slides](https://slides.google.com)
2. **File** > **Import slides**
3. **Upload** and select the `.pptx` file
4. Choose to replace or add to existing presentation

## Regenerate

From the project root:

```bash
python generate_user_manuals.py
```

## Contents

- **SAELAR**: NIST 800-53 assessment, AWS config, Chad AI, Risk Calculator, SSP Generator, BOD 22-01
- **SOPRA**: CSV-based assessment, Dashboard, ISSO Toolkit, Vulnerability Management, SSP Generator

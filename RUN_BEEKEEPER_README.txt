BeeKeeper — How to Run
======================

Option 1: From project folder
  .\run beekeeper
  or
  .\run_beekeeper

Option 2: "run beekeeper" from anywhere
  1. Add this folder to your PATH:
     [Environment]::SetEnvironmentVariable("Path", $env:Path + ";c:\Users\iperr\TestZippr", "User")
  2. Restart terminal
  3. From any folder, type: run beekeeper
